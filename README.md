# Política de Privacidade - Milvan

Este repositório contém a política de privacidade do aplicativo **Milvan**, um app educacional para alunos.

## Sobre o App

O Milvan é um aplicativo híbrido desenvolvido com Capacitor para facilitar o acesso de alunos a serviços educacionais. Inclui funcionalidades de login com câmera e autenticação biométrica.

## Política de Privacidade

A política de privacidade está disponível em: [privacy.html](privacy.html)

Esta página é hospedada via GitHub Pages e é usada no Google Play Console para cumprir os requisitos de transparência de dados.
