Política de Privacidade do Aplicativo Milvan

Última atualização: [16/04/2026]

O aplicativo Milvan ("nós", "nosso" ou "app") é desenvolvido para facilitar o acesso de alunos a serviços educacionais. Esta política explica como coletamos, usamos e protegemos suas informações.

1. Informações que coletamos

  • Câmera: Usada apenas para capturar imagens necessárias ao login e autenticação biométrica do aluno. As imagens não são armazenadas permanentemente nem compartilhadas.
  • Biometria: Dados biométricos (como impressão digital ou reconhecimento facial) são usados exclusivamente para autenticação local no dispositivo.
  • Dados de login: Informações básicas de acesso ao sistema educacional.

2. Como usamos as informações

  • As imagens da câmera são processadas apenas para verificação de identidade e login.
  • Dados biométricos permanecem no dispositivo e não são enviados a servidores externos.
  • Não coletamos dados pessoais além do necessário para o funcionamento do app.

3. Compartilhamento de dados

  • Nenhum dado é compartilhado com terceiros sem seu consentimento explícito.
  • As informações ficam armazenadas localmente no dispositivo.

4. Segurança

  • Usamos criptografia para proteger dados sensíveis.
  • Não armazenamos imagens ou dados biométricos em servidores externos.

5. Direitos do usuário

  • Você pode excluir dados locais a qualquer momento desinstalando o app.
  • Para dúvidas, entre em contato: [milscodgos@gmail.com]

6. Mudanças nesta política

Podemos atualizar esta política. Notificaremos sobre mudanças significativas.
